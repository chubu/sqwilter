# Sqwilter

## Installation

[Sqwilter - Chrome Web Store](https://chrome.google.com/webstore/detail/sqwilter/jbcjlklkgcpnellfgcogampdnlncmlla)

## Usage

1. Open https://app.sqwiggle.com/_settings/av
1. Select **Aspect** or **Filter**
